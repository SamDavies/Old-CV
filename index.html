Unibuddy Projects

Unibuddy GraphQL API - I lead development of Unibuddy’s GraphQL API and event driven micro-service architecture. The micro-services communicate using kafka and deploy continuously using Circle CI.

Kafka Streaming Analytics - I lead development on a real time steaming analytics system using an event based architecture. The system was used to track click streams on the Unibuddy platform. The system is massively scalable using kafka as a message queue and processing data in background jobs. The kafka streams API aggregates the data on the cluster and stores the data an read optimised query service. I set up unit tests, integration tests and acceptance tests, all continuously deploying on Circle CI.

WebSocket Chat API - I built the Real time chat API at Unibuddy with web sockets scaling to thousands of concurrent users and persisting to MongoDB.

React Native App - I lead development of the iOS and android React Native mobile app, including automated UI tests and continuous deployment with the fastlane tool.

React Web Apps - I was the primary frontend developer for the suite of React client apps at Unibuddy. This included the user management analytics dashboard and the main consumer facing web application. These were all built with react all connecting to the GraphQL API through the Apollo client.

Facebook Messenger Bot - I also lead development of the conversational Facebook bot, connecting the Unibuddy chat API directly to the Messenger Platform.

Snaptale - Photo Sharing Social Network

Snaptale is an ios app and a social network which helps you find the best student events and night clubs. For my 4th year honours project, I implemented a newsfeed ordering algorithm inspired by Facebook’s EdgeRank and a recommender engine to predict future ‘Tales’ using Spark. In the same way that Netflix recommends movies, Snaptale is able to learn what Tales a user likes and push these to the top of their newsfeed.

SmartLock

This project has a dedicated website. A team project to design and implement a smart door lock for the IoT. The door can be remotely locked and unlocked via the smart phone apps, whilst still allowing the use of a physical key. You can enable your friends to lock and unlock your Smart Locks, all controlled through apps for iOS, Android and Web. Machine learning techniques are being used to classify malicious entries.

This project is now being converted into a native iOS app.
