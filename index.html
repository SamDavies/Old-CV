GradeHub

My grade prediction calculator, developed this year, soared to success amongst students with over 2000 user sign ups and even becoming the topic of a Tab Article . Its popularity was owed to the slick, intuitive interface and use of cutting edge technologies like the rising React JS framework.

Snaptale - Photo Sharing Social Network

Founder and creator of the social network Snaptale, available on the iOS app store. Users can create events (things) and share images.

Snaptale helps you find the best student events and night clubs with the potential to expand beyond the student target market. It is currently available on the iOS App Store.

For my 4th year honours project, I implemented a newsfeed ordering algorithm inspired by Facebook’s EdgeRank and a recommender engine to predict future ‘Tales’ using Spark. In the same way that Netflix recommends movies, Snaptale is able to learn what Things a user likes and push these to the top of their newsfeed.

ORBIT - Factset Online Report Builder Interface

My internship at Factset in London was to design a web interface in Angular JS, allowing non technical product developers to generate data reports. This replaced many steps in the development of all reports at Factset by highly automating the process. I also pioneered behaviour driven development in the company, providing a presentation and tutoring the engineers. They have since offered me a full time position as a Software Engineer.

Machine Learning Deep Neural Network

The aim of the project was to train multi-layer neural network classifiers and convolutional network classifiers to classify handwritten digits using the well- known MNIST dataset and Numpy. This was then written up as an iPython notebook. The project description can be found on the University of Edinburgh's course website.

SmartLock

This project has a dedicated website. A team project to design and implement a smart door lock for the IoT. The door can be remotely locked and unlocked via the smart phone apps, whilst still allowing the use of a physical key. You can enable your friends to lock and unlock your Smart Locks, all controlled through apps for iOS, Android and Web. Machine learning techniques are being used to classify malicious entries.

Picanmix - Dating Web Application

Created a dating website using the Flask framework for Python. Automated a comprehensive test suite and a coverage analyser. Features included an instant messenger, secure login system and complex choice logic. Awarded a mark of 93% for an excellent piece of software.

This project is now being converted into a native iOS app.
